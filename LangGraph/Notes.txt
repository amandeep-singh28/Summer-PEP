****Type Annotations****
We only have to work with python in data science. But we have problems like this
    (i)It does not have static type of the variable(We don't write the type of the variable)
        year = 2004
        Logically it must contains numeric only, but what if someone changes it to:-
            year = "2010"
        Python will change it.
    Solution:-
        Enforce python to be in the declared type classes.
        We will take use of the library:-
            from typing import TypedDict # it will not enforce the rules
            
        movie = {
            'name' : str
            'year : int
        }
        We will create a class like this:-
            class Movie(TypedDict):
                name : str
                year : int
            movie = Movie(
                name = "Minions",
                year = 2026
            )

****Introduction to LangGraph****
The whole idea of LangGraph is to create graph like structure.
It is built on top of LangChain.
Concept of chains, but using chains you can create simple workflow.
If you want to create complex chains LangChain is not helpful.

-> Key elements in LangGraph
(i)Node -> A Node is simply a function or an operation that performs some task.
(ii)Edge -> Which connects two nodes
(iii)State -> Interconeected data sctructure. We pass this data structure to the node so that the node can have the relevant information
For example:-
    Graph for a specific student(Student name and roll no is something which will be req in all the nodes, we will keep this information in this state variable)

    Lets say, if we are trying to create a whole report card for the student, first node is repsonsible for "aggregating the marks", 2nd node is repsonsible for the "extra curriculars". Now based on sum of this two nodes we want to categorize the student is in Category B or Category A.
    So this information is something that we want to travel all across the nodes, so we will keep it in the state variable.

(iv)Conditional Node -> Chooses the next node based on a condition.
(v)Start / End Node -> Ending nodes can be multiple
