****Lang Chain****
It is an open source framework which is used to develop LLM(Large Language Models) applications
- Fundamentals
- RAG
- AI Agents

For example:
There is a PDF(E Book), what it does is This is machine learning handbook(Every concept of ML is there). We want to chat with this pdf and ask some questions like:- What is linear regression, Summarize page number 20, etc.
When LLM was not there it was very difficult to make this type of application.
There are two main challenges in this- We want our model to understand what the user is asking(English -> NLU), Context awareness.
