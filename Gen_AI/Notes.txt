****Chain****
What are we doing?
Pipeline: Input → Prompt → Model → Parser → Output
We first provide an input to the application.
That input is converted into a prompt (either a Single Message Prompt or a Multi-Message Prompt).
The prompt is then sent to the LLM (Model).
The model generates a response.
The response is passed to an Output Parser, which formats or extracts the required information.
Finally, the parsed result is displayed as the output.

Here are the steps in a short and easy-to-remember format:

(i)Import the required libraries.
(ii)Create the Prompt Template (ChatPromptTemplate).
(iii)Create the Model (ChatGoogleGenerativeAI).
(iv)Create the Output Parser (StrOutputParser).
(v)Invoke the Prompt using chat_prompt.invoke() to generate the final prompt.
(vi)Invoke the Model using model.invoke() to get the model's response (AIMessage).
(vii)Invoke the Parser using parser.invoke() to extract the text from the model's response.
(viii)Print the final output.

****Conditional Chaining****
RunnableBranch is used to implement conditional chaining in LangChain. It allows you to execute different chains based on a condition.

from langchain_core.runnables import RunnableBranch

conditional_chain = RunnableBranch(
    (func1, chain1),
    (func2, chain2),
    default_chain
)
How it works
Each function (func1, func2, ...) acts as a condition and must return either True or False.
RunnableBranch evaluates the conditions from top to bottom.
As soon as a condition returns True, the corresponding chain is executed.
The remaining conditions are not checked.
If none of the conditions return True, the default_chain is executed (if provided).
Example Flow
            Input
              |
              |
     -------------------
     |                 |
   func1?           False
     |
   True
     |
   chain1
     |
    Output

If func1 = False
        |
      func2?
     /     \
  True     False
   |          |
chain2   default_chain
   |          |
 Output     Output
Key Points
RunnableBranch works like an if → elif → else statement in Python.
Conditions are evaluated sequentially.
The first matching condition determines which chain is executed.
A default_chain is optional but recommended to handle cases where no condition matches.

If we have to work with multiple files, than we can use DirectoryLoader. We have to store all of the files in that Directory. It will start loading recursively.
  Create a folder, and store 2-3 pdfs and use this syntax:-
    from langchain_community.document_loaders import DirectoryLoader
    from langchain_community.document_loaders import PyPDFLoader

    loader = DirectoryLoader("Data Folder/", glob="**/*.pdf", loader_cls=PyPDFLoader, show_progress=True)
    documents = loader.load()

    print(f"Number of Documents: {len(documents)}")

    for idx, document in enumerate(documents, start=1):
        print(f"\nDocument {idx}")
        print("Content:\n", document.page_content)
    print("Metadata:\n", document.metadata)
